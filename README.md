## المميزات

### المصادقة والحسابات
- تسجيل الدخول بالبريد/الهاتف
- التحقق برمز OTP
- تسجيل الدخول بـ Google/Apple
- ملفات شخصية قابلة للتخصيص

### الشبكة الاجتماعية
- خلاصة منشورات (Feed)
- إعجابات وتعليقات
- قصص (Stories) تختفي بعد 24 ساعة
- متابعة/إلغاء متابعة المستخدمين
- إشعارات فورية

### المتاجر والأعمال
- تسجيل عمل تجاري مع مستندات
- صفحات أعمال تفصيلية
- إدارة الخدمات والأسعار
- نظام تقييمات ومراجعات
- دليل الأعمال والمحترفين
- بوابة السيدات (Ladies Gate) لقطاع الجمال والعناية الشخصية

### الحجوزات
- حجز مواعيد مع الأعمال
- إدارة الحجوزات (تأكيد/إلغاء/إكمال)
- تصفية حسب الحالة

### التواصل
- محادثات خاصة
- مجموعات (Groups)
- مشاركة الصور والفيديو

### الاكتشاف
- بحث شامل (مستخدمين + أعمال + منشورات + مجموعات)
- تصنيفات واقسام فرعية
- اقتراحات ذكية

### لوحات التحكم
- لوحة تحكم العمل (إحصائيات + حجوزات + خدمات)
- لوحة تحكم المحترف (ملف احترافي + إحصائيات)
- لوحة تحكم الإدارة (مستخدمين + أعمال + بلاغات)

## التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|----------|
| Next.js 16 | إطار العمل |
| React 19 | واجهات المستخدم |
| TypeScript | الأنواع |
| Tailwind CSS 4 | التصميم |
| Prisma 6 | ORM |
| MariaDB | قاعدة البيانات |
| NextAuth v5 | المصادقة |
| Zod | التحقق من البيانات |

---

## التشغيل المحلي

### المتطلبات
- Node.js 20+
- MariaDB 11+

### التثبيت

```bash
# 1. تثبيت التبعيات
npm install

# 2. إعداد قاعدة البيانات
# أنشئ قاعدة بيانات وأضف DATABASE_URL إلى .env.local

# 3. مزامنة Prisma
npx prisma db push

# 4. تعبئة البيانات الابتدائية
npx prisma db seed

# 5. تشغيل السيرفر
npm run dev
```

يفتح التطبيق على: **http://localhost:3000**

## هيكل المشروع

```
src/

    bookings/      # الحجوزات

  components/    # مكونات React
  lib/           # أدوات وقاعدة البيانات
  auth.ts        # إعداد NextAuth
  proxy.ts       # حماية المسارات
prisma/
  schema.prisma  # مخطط قاعدة البيانات
  seed.ts        # بيانات ابتدائية
public/
  logo/          # الشعارات
```
